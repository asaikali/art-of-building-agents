# 05 Booking

Phase 5 scaffold for the business meal planner.

## Purpose

This sample app will evolve into the booking phase that:

- confirms the selected option
- performs booking actions
- handles final booking-related follow-through

## What's here

| File | Purpose |
|------|---------|
| `BookingApplication.java` | Spring Boot entry point |
| `BookingAgentHandler.java` | Minimal `AgentHandler` backed by a bare `ChatClient` |
| `application.yml` | Local app and model configuration |

## Run it

```bash
cd business-meal-planner/05-booking
../../mvnw spring-boot:run
# Open http://localhost:8080
```
