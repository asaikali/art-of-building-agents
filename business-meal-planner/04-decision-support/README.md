# 04 Decision Support

Phase 4 scaffold for the business meal planner.

## Purpose

This sample app will evolve into the decision-support phase that:

- answers questions about shortlisted restaurants
- compares options
- helps the user choose what to do next

## What's here

| File | Purpose |
|------|---------|
| `DecisionSupportApplication.java` | Spring Boot entry point |
| `DecisionSupportAgentHandler.java` | Minimal `AgentHandler` backed by a bare `ChatClient` |
| `application.yml` | Local app and model configuration |

## Run it

```bash
cd business-meal-planner/04-decision-support
../../mvnw spring-boot:run
# Open http://localhost:8080
```
