# Workshop Slides — The Art of Building Agents

Slidev presentation covering all 5 workshop parts.

## Prerequisites

- Node.js 18+
- npm

## Setup

```bash
cd slidev
npm install
```

## Run (dev mode with hot reload)

```bash
npm run dev
```

Opens at http://localhost:3030. Edit `slides.md` and changes appear instantly.

## Presenter mode

Append `?presenter` to the URL: http://localhost:3030?presenter

Shows current slide, next slide, speaker notes, and timer.

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `→` / click | Next slide/animation |
| `←` | Previous |
| `o` | Slide overview |
| `d` | Toggle dark/light mode |
| `f` | Fullscreen |
| `?` | All shortcuts |

## Export to PDF

```bash
npm run export
```

## Build for static hosting

```bash
npm run build
```

Output in `dist/`.
